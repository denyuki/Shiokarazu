-------------------------------------------------------------------------------------


Need cogs, gears and inner workings for a minigame? Need some details for your steampunk machine?

Included a test scene with samples of the cogs animated using joints and physics.

24 unique cogs, 24 grayscale versions for re-colouring, 12 various fixings to further customise your cogs, the original Adobe Illustrator file.

3 Folders with 3 variations of each cog. Thin outline, thick outline and no outline.

All reasonable requests to improve this asset pack will be considered.


-------------------------------------------------------------------------------------


ORIGINAL FILES

-The cogs were created in Adobe Illustrator CC.  

-The originals can be found in 2 zip files.  One contains CC files, one has legacy CS5 files for back conpatability.  We highly recommend using the CC version if possible.  IF different versions are required let us know and we will update the asset.

-Both contain thick outline, thin outline and no outline versions of all the cogs.
